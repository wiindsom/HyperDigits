This page documents constructors and methods exposed to the returned HyperDigits class, documenting only the most-used/recommended methods.

All methods that take a number argument accept:

  • number
  • BaseHyperDigits table
  • HyperDigits

Operator notes:

  • Metamethods are implemented: +, -, /, *, ^, % and comparisons: <, >, <=, >=, ==

Percent notes:

  • Values created by fromPercent are converted into their decimal format.
  • Passing a percent into operations should work: HyperDigits.fromPercent(10) + HyperDigits.new(10) is equivalent to 0.1 + 10

fromString​

</>
Public.fromString(str: string) → HyperDigits

Parses a string into HyperDigits.

Supports:

  • Plain numbers: "123", "-123"
  • Scientific: "1e3", "1e-3"
  • Scientific-chain: "1e1e123"
  • Suffix: "1.23K" --Currently only works up to No and does NOT support saberSuffixes.

If parse fails, returns 0.
